# Esports Performance Analytics Dashboard

## 📌 Project Overview
Interactive dashboard analysing 3,123+ professional esports player records from Valorant Champions Tour 2024.

Designed to support coach-level strategic decision-making.

## 📊 Key Features
- Top 10 Player Ranking (ACS)
- Agent Usage Analysis (Treemap)
- Regional Performance Distribution
- Multivariate Scatter Plot (ADR vs ACS vs Headshot %)
- Interactive Filters (Agent, Team, Region, Player)

## 🛠 Tools Used
- Power BI
- Tableau
- Python (Pandas for data preprocessing)

## 🔎 Key Insights
- Regional role-based performance differences identified
- Correlation between damage output and combat score
- Clustered performance profiles across regions
